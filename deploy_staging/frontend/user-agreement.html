1. Общие положения

Настоящее Пользовательское соглашение (далее — Соглашение) регулирует отношения между владельцем сервиса ContractCheck (далее — Администрация) и пользователем сети Интернет (далее — Пользователь), возникающие при использовании веб-сайта contractcheck.ru.

2. Описание услуг

Сервис ContractCheck предоставляет услуги по автоматизированному анализу юридических документов с использованием технологий искусственного интеллекта. Результаты анализа носят информационный характер и не являются профессиональной юридической консультацией.

Пользователь обязуется не использовать сервис для анализа документов, нарушающих законодательство РФ. Администрация обязуется обеспечивать конфиденциальность загружаемых документов.

4. Оплата услуг

Оплата услуг производится в соответствии с выбранным тарифом через платежный сервис ЮKassa. Услуга считается оказанной в момент предоставления доступа к результатам анализа в личном кабинете.
